This project is to find the diameter in a given graph
------------------------------------------------------------------------

BFS (Breadth First Search) and finding the Diameter:
			Graph.java
			BFS.java
			Diameter.java

Graph.java - Used to create a graph for the input provided.
BFS.java - breadth first search traversal is implemented.
Diameter.java - Used to find the diameter in a graph. Returns a path between two farthest nodes in a graph.
